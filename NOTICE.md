# Notice

`research-figure-composer` is a Codex skill for generating editable SVG research figures.

The workflow references and adapts public, open-source ideas from the `nature-skills` project, especially the `nature-figure` approach to restrained scientific plotting, low-saturation palettes, editable vector export, and publication-style QA. The bundled implementation in this repository is self-contained and does not vendor the full upstream project.
